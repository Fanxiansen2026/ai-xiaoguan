💡 常见问题

问题1：API Key 未配置

→ 请先在主应用【后台管理】页面粘贴你的百炼 API Key 并保存。


问题2：API Key 无效

→ 前往 阿里云百炼控制台 检查 Key 是否已过期或被删除。


问题3：网络/CORS 错误

→ 阿里云百炼支持跨域，一般不会出现此问题。如果持续报错，请检查你的网络环境是否限制了对外访问。


问题4：主页面能正常但诊断工具报错

→ 诊断工具是独立文件，可能 Key 检测逻辑不同。请优先信任主页面的实际测试结果。
