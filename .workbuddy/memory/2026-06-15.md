## 技术笔记

- Worker 的 `startTime` 变量必须在 `/chat` 端点开头定义，否则 token 记录时会 undefined
- javascript-obfuscator 的 `selfDefending: true` 会导致代码格式化后崩溃，必须关掉
- Git push 被 GitHub secret scanning 拦截时，需要重置到远程状态重新提交（不能只删除文件，commit 历史中仍包含 secret）
- 语音按钮事件：直接在 HTML 中添加 `onclick` 比事件代理更可靠（混淆后函数名可能被改变，但 `onclick` 中的函数名会被 obfuscator 自动更新）
